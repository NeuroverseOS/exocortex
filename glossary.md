# Glossary

Terms specific to the exocortex project.

| Term | Definition |
|------|-----------|
| **Exocortex** | A structured external memory — a set of markdown files in a git repo that give an AI agent persistent context about its human collaborator. |
| **Attention** | The owner's declared focus for the current session. Singular. Only the owner edits it. |
| **Layer** | One of three context scopes: local (personal), org (shared company context), project (shared project context). |
| **Symlinked layer** | An org or project layer that exists as a separate repo, symlinked into the personal exocortex. One copy on disk, many readers. |
| **Intercognitive** | Thinking together — shared intersubjective reasoning between humans, or between a human and an AI partner. |
| **Conscientiousness augmentation** | The idea that AI's most transformative capability is helping people stay goal-oriented across time and context switches. |
| **Sprint** | At most a week of work. What we're doing right now, not where we're headed. |
| **Roadmap** | 3-month to 3-year view. Where the project is headed, what depends on what, in what sequence. |
